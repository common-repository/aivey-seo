== Description ==

Analyze posts and pages in real-time to ensure that the best SEO practices are implemented.

This plugin is currently in alpha development with the following features:

1. Adds "Meta Description" field to the Post and Page edit screens.
1. Adds meta description tag to HTML of posts and pages.
1. Scans Title field and warns when over 55 characters.
1. Scans Meta Description field and warns when outside 150-160 characters.
1. Scans Visual Editor and Text editor and warns of images without alternative text.

If there are any features you would like to see added next, please let me know. 

You can contact me at my website: http://alexjivey.com

== Installation ==

1. Upload aivey-seo.zip contents into your Wordpress "/wp-content/plugins" folder.
2. Activate the plugin in the Plugins section of the Wordpress administration panel.
